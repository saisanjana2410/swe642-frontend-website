## Overview
This project is a front-end website developed as part of SWE 642. It demonstrates the use of:
- HTML5
- CSS3
- Bootstrap framework

The website consists of:
1. A personal homepage
2. A Computer Science Department information page
3. A student survey form for prospective students

## Features
- Responsive design using Bootstrap
- HTML5 form elements (datalist, radio buttons, checkboxes)
- Client-side validation using required attributes
- George Mason University color scheme
- Hosted as a static website on AWS S3

## Technologies Used
- HTML5
- CSS3
- Bootstrap
- AWS S3 (Static Website Hosting)
